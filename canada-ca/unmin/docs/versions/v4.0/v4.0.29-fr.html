Quoi de neuf?

Version 4.0.29

Version 4.0.28

  • Mineur - Filtre - Ajout de l'interprétation des espaces comme un opérateur booléan (et, ou)
  • Mineur - Graphique - Ajout d'une configuration possible pour contourner la valeur de la cellule de donnée
  • Mineur - Ajout d'une liste de vérification pour l'accessibilité sur des appareils mobile
  • Information gababrit - Mineur - Changement dans le balisage du gabarit pour le gabarit de connexion, voir PR 8382.
    • L'élément <p class="btn"> a été modifié par <span class="mrgn-rght-tp">
  • Mineur (Majeur pour IE < 9) - Mise à jour vers la dernière version de jQuery 1.x et 2.x
    • Des changements dans le balisage est requis pour ceux qui doivent supporter IE < 9, voir PR 8371. IE 10 ainsi que les versions précédentes ne sont pas supportées par cette version de la BOEW.
  • Mineur - Flux - Ajout du support pour des titre en format XHTML definie dans un flux Atom
  • Développeur BOEW - Majeur - Modification du script de compilation
    • Introduction d'une vérification automatisé des espaces, l'indentation doit être maintenant faire à l'aide de charactère de tabulation.
    • Mise à jour de quelques version pour les dépendances de nodeJS.
    • Les développeur de la BOEW devrons peut-être supprimer le cache de npm ainsi que les répertoires "node_modules" et "lib" avant d'exécuter "script/setup" pour être capable de compiler avec succès.
  • Correctif d'accessibilité:
    • Menu mobile - Cas d'utilisation d'où qu'il n'y a aucun bouton de fermeture
    • Menu - Navigation par clavier pour les sous-sous-menus
    • Expiration de session - Texte de lien vide
    • Menu - Style pour la cible sur un item de menu qui n'a pas de sous menu
    • Menu - Couleur d'arrière plan en mode HTML simplifié
    • Accordéon (Basculer) - Navigation par clavier
    • Onglet - Navigation par clavier
    • Détails - Ajout d'un contour lors du ciblage
    • Mega menu - Navigation par clavier (#8382, GCWeb #1335)

Recommendation de mise à jour de balisage (Mineur)

  • Gabarit de page - Mise à jour de la version de jQuery
    • En bas de la page: <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.js"></script>
    • En haut de la page à l'intérieur des balises conditionnel IE 9: <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.js"></script>

Canada.ca (GCWeb):

  • Mineur - Retrait des préfix CSS IE < 11
  • Mineur (Majeur pour IE < 9) - Mise à jour vers la dernière version de jQuery 1.x et 2.x
  • Mineur - Prévention de l'impression du bouton "Signaler un problème sur cette page"
  • Mineur - Revision du balisage pour le profile ministériel - voir #1365
    • Retrait des classes CSS .h5 pour la section titré "Coordonnées" - Ligne 29
    • Utilisation de l'élément <strong> pour mettre en évidence les mots "Téléphone" et "Courriel" dans la section coordonnées. Ligne 32 à 33.
    • Les dernières nouvelles, le balisage utilisé pour la description associé utilise une meilleur sémentique - Ligne 52 à 53
    • Le balisage pour les items de la galleries a été revues. Tel que définie par bootlint: E013 (Les enfants des lignes doivent être des colonnes); E014 (Le parent d'une colonne doit être une ligne).
  • Mineur - Revision du balisage pour l'index de navigation locale - Voir #1365
    • Retrait du clearfix après la section "Services et information". Juste avant Ligne 51
    • Ajout de la classe CSS whtwedo pour la section "En demande". Ligne 56
  • Mineur - Déroulement de champs - Ajout d'une action configurable si aucun ifNone choix n'est fait.
  • Mineur - Corespondance d'URL - Conversion du symbole "+" en un caractère d'espacement
  • Majeur - Mise à jour du contenu pour le mega menu
  • Mineur - Correctif de l'espacement en haut (top margin) pour la page des sujets
  • Mineur - Ajout partiel de la page de long index
  • Majeur - Revision considérable de la page d'accueil de GCWeb
  • Mineur - Retrait de la classe CSS pagetag. Ceci à un effet pour les gabarits suivant:
    • Département et agence
    • Sujet
