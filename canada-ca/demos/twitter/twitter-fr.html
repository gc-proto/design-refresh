Example: Embedded Search Timeline

An embedded search timeline requires a twitter account to generate the query and widget id.
Tweets concernant "#boew"

Code

<div class="wb-twitter">
<a class="twitter-timeline" href="https://twitter.com/search?q=%23boew" data-widget-id="329071414877495297">Tweets concernant "#boew"</a>
</div>
